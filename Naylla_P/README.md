### Weather API
The weather widget is currently set to **DEMO MODE** to ensure it looks good immediately without an API key.
To use real live weather data:
1.  Get a free API Key from [OpenWeatherMap](https://openweathermap.org/).
2.  Open `script.js`.
3.  Replace `const API_KEY = 'YOUR_OPENWEATHERMAP_API_KEY';` with your actual key.
4.  Uncomment the fetching logic in the `fetchWeather` function.
